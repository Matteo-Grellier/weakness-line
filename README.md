# Projet desktop

Ce projet consiste à créer un logiciel de présentation comme powerpoint avec l'utilisation d'electron.

## Installation

Installer les dépendances :
```bash
npm install
```

## Création du package

```bash
npm run package
```

## Usage

```bash
npm run dev
```
